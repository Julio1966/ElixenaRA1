Logros de la Unidad 1 - Aplicaciones Web I4.0

En esta página, comparto los logros alcanzados en la Unidad 1 de Aplicaciones Web I4.0. Durante este periodo, hemos explorado y aplicado herramientas esenciales para el desarrollo web contemporáneo.

Puntos Destacados:

  • Dominio y aplicación de Express, un marco de aplicación web para Node.js, simplificando la creación de servidores web y la gestión de rutas y solicitudes HTTP.
  • Empleo de Dotenv para una administración segura de las variables de entorno en nuestras aplicaciones.
  • Desarrollo del frontend con Angular, un framework de código abierto que facilita la creación de interfaces de usuario interactivas y modulares.

En resumen, esta etapa nos ha capacitado para edificar aplicaciones web de alto rendimiento y calidad, haciendo uso de herramientas avanzadas y prácticas de desarrollo eficientes.
